== Description ==

This is an add-on for the [Restrict Content Pro plugin](http://pippinsplugins.com/restrict-content-pro-premium-content-plugin/). It does not function on its own.

This plugin will add support for limiting bbPress forums and topics to paid subscribers and / or subscribers with specific access rights. It makes it exceptionally easy to have members-only forums

Learn more about Restrict Content Pro at [Pippin's Plugins.com](http://pippinsplugins.com/restrict-content-pro-premium-content-plugin/)
